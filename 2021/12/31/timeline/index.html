闲言碎语-2021

2022-01-01

2021 年到来了，话说时间确实快，不敢想象，敬畏时间，多做有意义的事情。

电脑上用五笔输入法，手机上用拼音输入法，是我十多年的习惯了，曾经有几次都想统一成拼音输入法，无奈一直没有可靠的方案。

国内的拼音输入法其实很棒，但是考虑到国内公司毫无商业道德底线，随意监听用户信息，我在手机上一直用 Gboard 拼音，电脑上就是自造词库的 Rime 五笔。

虽然输入法不同，但是使用起来毫无违和感。五笔输入法最大的问题是有些字不知道怎么写，这导致打不出来。虽然可以临时切回拼音，但是还是稍微有点麻烦。并且有时需要反复输入多次才能把汉字输入正确。

“内卷”一词最早由德国哲学家康德提出，经历了多位学者的延伸与拓展，其主要强调发展模式的停滞不前。而当下，“内卷”更多意味着“白热化的竞争”。“内卷”在国内火起来是几张名校学霸的图片：有的人骑在自行车上看书，有的人宿舍床上铺满了一摞摞的书，有的人甚至边骑车边端着电脑写论文。现在很多高校学生用其指代非理性的内部竞争或“被自愿”竞争。

这两天尝试了将电脑上的输入法从Rime五笔转到Rime拼音，用了一下总感觉不如五笔来的爽快，有一次尝试失败。

双十一后买的Switch和健身环一直没怎么用，都是儿子玩耀西的手工世界，以后我得把健身环用起来了。今天还买了俩游戏：马里奥赛车8和马里奥奥德赛，还买了个手柄方向盘。

又一次站在了人生的十字路口，这次选择应该能决定接下来五年甚至十年的命运。

终于把娃教会还原魔方了，小孩子的学习能力还是挺强的。

关于密码管理工具，我选来选去还是觉得LastPass最好，不过我其实不用它自动填充，主要是记录一些账户。再说了，国内的很多App是压根不支持自动填充的。

制作了个MIUI主题，更新了三版，依然过不去审核。

储蓄卡选来选取，最终还是感觉招商银行比较好用。16位数字，App上功能齐全，想做什么一个App就能搞定了。

双十二买了个笔记本，安装了 Deepin 系统，最近用的比较多，问题也越来越多了，性能这么牛逼的 Yoga 14s 居然在线看视频都卡顿，是那种画面卡顿不是网速慢。或许这是因为 Deepin 优化不够好？有机会了我安装个 Ubuntu 再试试。

今天是我人生第一次有50万流动资产，虽然不是绝对的可流动，但还是值得记录一下。

最开始有点钱买房了，然后买车，然后买车位，然后家里建房，再断断续续往外借一些，一直手里可支配的钱都不多。

今天拿到公司的 N+3 补偿后，再加上手里的钱，算是能够支配 50 万了，但是过了五一这些钱就又要分开了。

今天骑行4个小时52公里，状态比前两周的骑行好很多。越锻炼状态越好，以后要勤加锻炼。

转：说阿里巴巴违反《反垄断法》，按照法律条文来看的话，不冤。但是按照同样标准来看，那个量级的公司其实没几个无辜的。

这个就叫“高标准立法、普遍性违法、选择性执法”。

怎么个选择性执法？那就看具体情况了，给的好处不到位，背后的靠山倒了，上头严查了，或者政治出问题了，那就该你倒霉了。

我最不爽的就是这个“高标准立法立规、普遍性违法违规、选择性执法”，这真的是屡试不爽，中国政府的这套标准真的是坏到极点了。

一次交住院费非常不好的体验，我把中信银行信用卡注销掉了。其实主要问题是出在医院，因为医院其他窗口能微信、支付宝，而住院费窗口却不能。后来我在云闪付上开通了中信银行信用卡的NFC闪付，一直提示密码错误，一气之下就把信用卡注销掉了，反正这张卡我基本上已经不使用了。

刚看到新闻，说是国内73家影视机构联名维权，要禁止用户对其具有版权的影视作品进行剪辑、切条、搬运、传播等。

国内拍的电影、电视剧越来越差，追完全剧真的是浪费时间，在抖音上看剪辑的精华部分就成了很多人的首选，毕竟有时和别人聊天难免会提起热门剧，自己没看过还真搭不上话。

有些影视剧的解说也很不错，我在 YouTube 上关注了一些，十来分钟看完一部长达两小时的电影真的很不错。

前几年是打击个人电台FM里有版权的背景音乐，现在又是短视频，这种真的是正确的版权方向么？未来是不是会涉及更多领域，进而阻碍创新呢？

正好用这笔钱验资开了海通证券的科创板、港股通、融资融券和期权，虽然我暂时用不到这些权限，但是提前解锁也没啥不好的。

越来越觉得没必要在支付宝上买基金了，开个证券账户，然后买个场内基金，设置好止盈条件，严格执行，小钱还是挺容易赚的。

以前玩博客时，我写文章的时候远远小于折腾博客系统的时间，现在做投资的时间又少于研究投资工具的时间。之前一直想要解锁证券和期货账户的所有权限，现在全部解锁了，又觉得有点空虚。投资是伴随自己一辈子的事情，但是现在却感觉无从下手了，钱就那么多，买买就没钱了，又不想频繁交易。

2016年开始主用工商银行到2021年年初也有四年多了，其实所谓主用也就是平时也就转转账、设置工资卡。前段时间我花了点时间把自己的需求理了理，最终决定主用招商银行储蓄卡了，招行优点：16位卡号，App转账额度高，有那种可以直接支付使用的理财产品。最终我把我的证券、期货、理财账户全部换成招商银行了，目前使用两个月了，体验还算可以吧。

今天在孩子脸上打了几下，把孩子脸打肿了，我特别特别后悔，心里内疚死了，真想将孩子脸上的伤转移到我脸上。唉，这对我来说是一次很大很大的教训，以后坚决不能用暴力了。

暴力教育真的是一件非常非常差劲的教育方式，孩子身上疼，家长心里痛，唉，我太失败了。

越来越欣赏不了 Android 的设计了，尤其是 Android 12，直接把我劝退了，从此再也不想买个原生 Android 系统的手机了，老老实实的买国内的手机。

今天发现我使用的一个国内金融App升级后不需要存储权限了，现在就剩招商银行App还需要存储权限了，垃圾招行越来越不给力了。

中国舆论用“内卷”形容这一现象——一个领域因过度竞争，导致人们相互倾轧、内耗的状态。

典型的剧场效应，本来大家都坐在座位上看戏谁也不影响谁，突然有人为了爽或其他原因突然站起来了，你不跟着站起来就无法看戏，然后只能大家全站起来看戏。

目前我所使用的国产应用包含传统金融类App已经全部不强制索要存储权限了，这个真是值得记录的事情。我原想着需要等到今年11月份才能实现，没想到今天就实现了。

转：当下是效率转向公平的一个转折点。效率过度极致会内卷，会失去活力。物极必反。

先富没带动后富，反而享受了很多红利比如房地产市场的大涨。慢慢的很多人不愿意再给资本家做燃料了，燃烧自己成就别人不是良性循环。

大公司也是时候该为社会公平性负起一部分责任了。

最近想改变一下生活方式，尝试多听一些音乐看看能否帮助我改变一下，入手了299元的红米无线耳机，对我来说真的贵啊。

十年没用 Windows 了，最近发现 Windows Terminal + WSL2 Ubuntu 真的是绝配，特别特别好用。我对系统的需求其实就是终端和浏览器，如果软件生态再好一些就更好了。

中国的疫苗接种真是特色“自愿”接种，不接种很多地方无法进入或者进入时要各种登记，反正就是逼着大家去接种，有点“即当婊子又立牌坊”的意思。

我都服了我自己，钱没赚到，App却装了一堆：期货的易星，证券的e海通财、银河证券、同花顺、雪球、微牛证券、富途证券。

突然特别不喜欢kubernetes里的kustomize这个工具，真的是太乱了，不如直接写yaml方便呢。

我的投资渠道现在终于理顺了：

场外基金用支付宝和中欧财富，前者用来零散的买基金，后者定投「中欧超级股票全明星」投顾组合。

场内证券、基金、期权等日常交易用海通证券，稳健、长期的投资用银河证券。

国内期货就用创元期货了，平时交易很少，机会合适了就交易一把。

港美股用微牛证券，富途证券里的股票我找机会转移到微牛。（因为微牛证券免佣）

银行理财我用招商银行和平安银行，将来看看是否有机会合并到一起吧。

基金投资：

  1. 不择时定投“中欧超级股票全明星”投顾组合

  2. 择时买卖双创50、沪深300和中证500指数基金

  3. 追踪热点行业主题基金

股票投资：

  1. 港美股随缘

  2. A股以长线为主，ST股票为辅

最近海通证券账户市值浮亏一步步扩大。。。

看到一篇文章说神雕侠侣里金轮法王收郭襄为徒是想和她双休。。。金庸的小说里脑洞大开的事件太多了，还有天龙八部里的扫地僧是李秋水等等。

微信公众号当时内测信息流时我就觉得它们要往里插入广告了，现在信息流里到处都是广告，已经无法看了。

这次出差，来回都是用的工商银行的白金卡进的贵宾厅和休息室，还算满意。用卡还是得靠四大行，商业银行总是耍猴，什么次数限制啦，权益要抢啦。

昨天晚上有一点点失眠，在一个群里说了好几个自己生活单调的句子。

白天工作，下班回家已经晚上八点多了，吃个饭然后被其他小事稍微一磨蹭就十点了，该睡觉了。洗衣做饭基本都不用我动手，但是依然觉得时间不够用。

偶尔加班到很晚，回来后直接睡觉。夜生活是压根不存在的，就连晚饭后小区楼下散步都不存在的。

周末更是就想呆在家里，打开电脑学习一些工作中要用到的东西，以前那种一劳永逸的日子早就不存在了，现在得不断的学习学习再学习。偶尔花费十几分钟看个动漫，要么就是躺在床上或坐在沙发上发呆。

农村孩子出来打拼，没有家底，上有老下有小，根本松懈不下来，想去做一些事情，也想去改变很多事情，但是这些都是以多花钱或少赚钱为代价的。

感觉这个社会从2000年以后就变了，大家都开始特别忙碌了。我记得小时，村里人外出打工，过年回家呆将近一个月，暑假呆一个月，春收、秋收呆一两周，现在呢，基本上一出去就是一年了。

社会财富越来越分配不均，富人凭借自己的关系或能力会变得越来越富，穷人只会越来越穷，社会体制越来越健全，条条框框越来越多，一切的一切看似平等其实主要是服务于富人的（整个社会资源有限，大家竞争资源还是要靠财富，像最近一直在推进的教育公平，这个对富人压根无效，只能减少穷人间的相互倾轧但是并不能从根本上改变什么），穷人阶级跃升越来越难。

想给家里老人买手机，现在可选择的太少了，或者说太多了。本来想首选小米的，但是小米功能太多太复杂了，比如短信app，设计的太复杂，老人根本不会用。

昨天，也就是国庆假期的前一天，我开车回山东老家，高速路上车辆很少，出行真的太便利了，唯一的遗憾是高速过路费只在节假日限免。

用了将近一年的 Deepin，发现我最终还是得回到 Ubuntu 的怀抱，Deepin 里的软件版本太低了，对新硬件的适配太慢了，我已经忍受了一年的小音量了。下周 Ubuntu 发布后就换过来，然后再安装 DDE 或 Cutefish DE。

折腾了一圈，感觉还是 Deepin 相对稳定并且适合我使用，到底是安装 Deepin 还是 Ubuntu DDE 呢？

今天给亲戚电脑安装Java以便能其使用公司的ERP系统，主流浏览器早已不支持Java，最后无奈找了 Firefox 50 32位版，刚使用一会儿，关闭再打开，发现自动升级了，然后就不能用 oaj2se 了。

刚刚看到两个概念：『木桶理论』和『长板理论』，前者是说能装多少水由最短的木板来决定，后者是说能发展到什么程度是由最长的木板决定。

现代社会，个人需要的是『长板』，把自己的长处发挥到极致远比补足短处获得的收益多。

在知乎上看到了段话：35岁失业的本质，是中国产业升级失败的结果。一个劳动者最能输出的年龄在55岁左右。如果产业总是徘徊在低端重复的水平，是不需要55岁熟练的经理，工程师的。与之对比一下，你看看做官员，是没有35岁失业的，因为官员领域人家是成熟的，广泛需要能力强，经验丰富的老手的。另外，炒股行业也不存在35岁失业的情况。大致可以认为，行业幼稚，而从业人员没有认清其幼稚本质，胡乱深耕，是35岁悲剧的主要原因。

纯碱这个品种涨的猛跌的更猛，被了几次保证金，扛下来了，小赚一把平仓了。我的心态还是不够稳，玩期货还得多练练。

每次喝酒或者喝咖啡就会失眠，以后得注意一下了。

公司 Windows 10 系统上很多软件不让安装，也不能使用 Linux 系统，而我之前在 Windows 上的解决方案是 WSL2 + Ubuntu + Windows Terminal ，现在完全无法使用。

Kafka 的文档真烂，这么大众的开源项目，文档这么差。。。

淘宝买东西不管用啥支付方式，付款后总会跳出同意花呗新协议的窗口，太恶心人了，我最终决定关闭花呗了。

我有点 Naive 了，太把自己当回事儿了，工作中除了能力以外还有很多东西。

不知道从哪天开始，我突然不追美剧了，有三个年头追了，或许能打动我的剧集越来越少了，或许是由于我看剧集讲解的缘故。
