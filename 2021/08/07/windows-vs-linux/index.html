Windows vs Linux

2021-08-07

用了一个多月的 Windows 10 系统，让我对 Windows 有很大的改观。

自从工作后，到现在十年多了，我一直都在用 Linux 和 MacOS，无论是家里电脑还是公司电脑我从来不用 Windows。

我一直比较喜欢程序 Web 化并且坚信未来一切都会基于 Web。工作中主要使用的两个程序是浏览器和终端，而 Linux 能完全满足我的需求。

我工作的前五年一直用的是 Ubuntu Linux，很顺手特别喜欢 Ubuntu Unity 桌面环境，后来 Ubuntu 转向了反人类的 Gnome3, 于是我就投入了 Deepin Linux 的怀抱（家里电脑）。

我工作的后五年一直用的是 MacOS，公司推广员工 Macbook Pro，于是我就也申请了，我是非常不喜欢 Macbook 的键盘，超级难用，我是强行将 Maacbook 的键盘改成我 Linux 下的使用体验，换了不少第三方软件，各有各的 Bug，就这样一直用了五年。当然主要使用的还是浏览器和终端。

在去年年末，我入手了联想的 Yoga 14s，挺好的笔记本，入手之后我就安装了 Deepin Linux，使用了半年时间，不得不说，Deepin 的界面确实很棒，但是稳定性实在太差了，小问题不断，还有超级大问题比如触摸板无法使用、视频播放器没画面，这种0级Bug居然能一个月甚至半年不修复。。。

换了新工作之后，公司给配置的是普通笔记本，我一开始是自己安装了 Deepin Linux（是的，虽然它很不稳定，但是我还是安装了），工作是没关系，但是团队协作和会议程序不支持 Linux，没办法，我只好换回了 Windows，并且发现了 WSL + Windows Terminal 这种超级组合。

Windows Subsystem for Linux + Windows Terminal 真的是很牛逼的组合，直接让我获得了仅次于 Linux 终端的使用体验，再加上 Windows 的应用程序生态，我觉得真的是很棒了。

现在公司电脑已经将 Deepin Linux 卸载了，在家也主要使用 Windows 了。用户真的是会用脚投票的，Deepin Linux 是真的太不稳定了，而 Ubuntu 的 Gnome3 桌面又是那么的反人类，我没得选了。

真没想到，毕业十年后我再次投入 Windows 怀抱，给我带来了不少惊喜。Windows 11 会有 Windows Subsystem for Android，这个真的是太棒了。

写着写着，写成流水账了。。。
