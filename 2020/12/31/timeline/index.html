闲言碎语-2020

2021-01-01

2020 年到来了，话说时间确实快，不敢想象，敬畏时间，多做有意义的事情。

在装修中被收了“智商税”不是因为你智商低，只是商家利用信息差蒙蔽了你，你要做的不是懊恼自己笨，而是要掌握足够多的信息，比如仔细阅读我这个回答。https://daily.zhihu.com/story/9718395

从2017年国庆之后玩股票到现在，我还没搞懂什么是PE、PB、PEG等等概念，还有KDJ和MACD等等，看来需要好好学学了。

刚把回家的票给退了，今年过年不回家了。

我内心是很矛盾，家里全是老人，让他们来西安他们不愿意，而我在家也没什么朋友，出门很不方便，所以我也不太愿意回家，但是我依然把回家的火车票给买了。

上午和同事聊天，因为传染病的问题，我同事把回家的票给退了，我也跟着退了。内心有点不安，过年不回家和家人团圆，尤其是我家全是老人的情况下。

代沟可能真的很难跨越，我一直都弄不懂家人心里想的什么，明明来西安过年是多么好的一件事，现在农村过年已经没什么年味了，很没意思。城市里过年反而越来越热闹，留在城里过年的人也越来越多了。

孩子想孝敬父母，而父母又不想打扰孩子，然后带来的结果就是两者都不是很舒坦。

刚刚我四岁的儿子跑过来问我：“爸爸，你知道为啥女生爱男生吗？” 可把我笑的肚疼。

国内的非营利组织都这么垃圾吗？红十字的名声彻底被那些狗日的给玩坏了。能不捐款就不捐款，一定要忍住。

现在越来越没耐心了，看影视剧都是1.25或1.5倍播放，很多时候还只看解说懒得看剧了，也不知道这个是好是坏。

NodeJS 的 Module 真是个大垃圾啊，一大堆东西无法用。。。

失眠了，一夜没睡着。

今天在群里聊了聊各自的文件备份方案，然后我下决心将百度网盘卸载了，百度网盘可是最近八年多一直在我手机上安装的应用。

虽然有点不舍，但是百度网盘下载太鸡肋了，一个几兆的文件下载三五分钟都下载不下来，实在让人忍无可忍。

上周终于下决心给我的 redmi k20 pro 刷了类原生系统，话说自带的Google服务体验就是棒！唯二不太好的地方就是续航差点和无法使用NFC门禁卡。

最近美股大跌看出来了一件事：场内的跨市场交易（QDII）太不靠谱，场内压根不按净值来，虽然场外手续费会掉层皮，但是好在能跟着美股趋势，自己容易做判断。

我的投资太分散了，最近在慢慢的集中到一起。平安证券里的股票和基金今天全部转出来了，融通基金里的钱上个月转出了一半，银行理财除了平安银行之外其他全转出来了。

现在我的投资国内集中在支付宝、且慢基金、海通证券、长江证券、平安银行，国外集中在富途证券、工行香港。尽量不再开通其他同类渠道了，实在太麻烦了。

买基金还是首选支付宝，赎回第二个工作日就到账了。平安证券里的基金要第四个工作日才能到账，明显是坑爹。且慢基金里面赎回也是第二个交易日，估计平安证券占用了我赎回的钱做了什么事情，还有就是平安证券里的基金交易费用貌似也比较高。

还是经验不足，打着打着没子弹了，现在眼睁睁着看着股市在跌，而我却无能为力。

前段时间还想买 Mini PC ，看了 Intel 的 Nuc 和 Dell 的 OptiPlex Mini ，没有新款，现存的电源适配器太大，USB 接口也不令人满意，而 AMD 的 Mini PC 也一直没动静。

最近了解了一下笔记本，非常喜欢 Chromebook，但是目前依然没有能令我十分满意的，造型好看的配置不行，配置好的造型不好。我现在想买个 Chromebook 在家里外接键盘、显示器用，也能单独带出来用。Lenovo 的 IdeaPad Duet Chromebook 造型让我十分满意，就是配置太差了。

这次股票跟热点让我吃了不少亏，三支ETF（新汽车、电子ETF、创精选88）合计得赔35%以上。其实完全没必要参与新ETF的认购，市场上已经有很多同类ETF了。

总会看到有人说：“Mac 就是好用，有钱就买 Mac”，但是根据我这些年的使用经验来看，Mac 的工作效率不如 Ubuntu Unity。

比较可惜的是 Ubuntu 放弃了 Unity 而转向了 Gnome3，自从 Gnome3 推出到现在我都觉得它是个反人类的东西，我是不会使用它的。

最近在了解 Chromebook，可以运行 Android app，感觉这个东西也许是才是未来。我的使用搭配可能会是一个微型 AMD64 主机做软路由和工作站，Chromebook 做桌面端。

今天发工资了，有点失望，因为到手的钱还没去年多，关键我去年还涨工资了。一直努力工作，牺牲掉了很多生活时间却得到了这样的回报！

我最近这几年一直在还款，买房首付借的钱好不容易还清了，然后又买了车还车贷，然后买了停车位，接着又是家里集资建房我又借了九万块钱，这个月算是把除房贷和车贷（还剩俩月）的其他借款还清了。

接下来还要攒钱，因为投资失败的钱也需要补上，这是我爸妈的钱，暂时没有啥期限，每个月少存一点，慢慢就补上了。

工作和生活压力挺大的，想买个电脑，现在都两年过去了一直没买，一来是因为电脑对我来说是伪需求，再一个就是因为钱。

前几天看到一个很喜欢的Chromebook，价格不贵，等正式上线了我可能会买一个。

一直觉得娃太小没法玩平衡车，今天玩了一下他表哥的平衡车，不到五分钟就学会了，给我带来了很大的惊喜。然后我就在京东买了个小米平衡车，明天到货，让他好好玩玩。

摆脱微信第一步就是不再保留里面的聊天记录，我之前微信里一直保留着最近六年的聊天记录，期间换了三个手机，真不容易。最近我几次卸载安装微信没再保留聊天记录，然后发现其实也没什么，以前一直认为重要的聊天记录其实并不是真正的重要，有些东西过去就过去了，没必要为了将来『或许』能用到而保留。

和一部分老外工作真能把人气死，甩锅能手，他们想的不是设计是否合理，而是以自己做的工作少为前提，不管给客户或别人带来多大麻烦。有时真不想和他们一起工作，能把人气死。也许工作只是人家生活的一小部分，所以能少做事情就少做，加上他们的语言优势，越想越觉得委屈。

今天下午浏览了一些十年前保存的图片，发现我上大学那会儿是真的很愤青，还好工作之后不再愤青了，人总是在成长。对国家对政府再失望也不能对自己失望，努力过好自己，慢慢影响身边人。

今天把平安证券销户了，2016年前在京东股票App里开通的，当时什么都不懂就开通了，然后和炒股有关的骚扰电话就开始多起来了，一直到现在都没消停。2019年开通了海通证券，佣金比较低，一直在用海通。今年也顺便开通了长江证券，当时的想法是一个玩长线一个短线，现在想想其实必要性并不大。

我一直都是百度网盘的坚定拥护者，但是我一直都是免费用户。虽然之前有过两年百度网盘VIP会员，但那是因为参加活动以很少的钱买到的。我其实一直没有付费的需求，毕竟我使用百度网盘也就是保存一些图片和视频，这都是基本的免费功能，而空间容量也在前些年网盘大战时扩大到了很大。也许这就是百度网盘的尴尬处境，免费功能基本都满足用户了，而又没什么可扩展的付费场景，尤其是大家把百度网盘作为一个一次性的下载工具，下载完就走了，没法留住用户，进而无法从这些用户身上获利。这真是个一直投入却基本没产出的产品，能坚持到现在确实非常不容易了，希望百度网盘能尽快找到一个比较好的营利途径，加油！有一点是我非常不喜欢的，就是百度网盘会自动将里面的『不良内容』给换成8秒短片，这个非常让人不爽，但是这个貌似又是百度网盘在特殊国情下不得不做的事情。

刚刚突然想到我最近十来年一直在用 Google Voice 注册各种服务，万一哪天 Google 要收回 Voice Number，我直接就完蛋了，有太多太多的服务绑定的是 Google Voice 了，这个始终不如弄个实体号码靠谱。

在群里聊起这事，然后一个群友说是可以花30刀买永久的，然后我就买永久了，希望 Google 的永久是真的永久。

持有的一手中国平安股票分红了130元，值得纪念一下。

车贷历时两年，现在终于还清了，房贷还得十几年，期待还清的那一刻。

今天中信信用卡额度给自动提升到10万了，到上次为止我六张卡中有四张10万额度的了。消费能力跟不上，空有额度转无法转化成收益。

打脸了打脸了，我的 K20 Pro 有刷成类原生了，MIUI12 我实在用不习惯。

周六回媳妇家，发现她给娃买了个魔方，然后我突然就想好好学习一下魔方还原方法，然后一发不可收拾。我觉得这个魔方作为减压工具还是挺不错的。

最近心态不好，影响到工作了，和同事开会英语沟通能力明显下降了，得调整一下心态了，希望能尽快回到良好的状态。

最近迷恋上魔方了，然后把二、三、四、五阶魔方凑齐了。五阶魔方准备过段时间再玩，听说会玩三、四阶魔方之后其他的就都不是问题了。

K线图还分『不复权（除权）』、『前复权』和『后复权』，我以为我买在相对低点，用前复权一看，还是挺高的，失误，失误。。。

一直在使用一个『有道词典』的扩展，非常符合我的要求，由于其使用的旧 API 已经停止接受新的请求了，于是我就找到源码修改了一下适配新的 API ，现在再也不会出现『请求太频繁』的错误提示了。https://github.com/chenzhiwei/chrome-youdao-dict

今天还顺手把博客使用的 Hexo 升级到了 5.0.0，这个版本对我来说最喜欢的功能就是把主题配置从主题目录移动到了项目跟目录，这样使用别人主题时就非常方便了。

一直都不怎么喜欢美团，王兴从创业就开始抄，一直抄到现在，前些年骂『狗日的腾讯』，现在跪舔并认腾讯当干爹，想想就可笑。

以牺牲消费者/用户的利益而做出的改变太不尊重用户了，希望大家能有更高的觉悟，让这类公司无法生存。

他所说的微信比淘宝手续费低我感觉应该特指美团本身，但是他却只说了一半事实，误导大众。

最搞笑的是他还说淘宝也不支持微信支付。。。支付平台太多了，京东也不支持支付宝呢，什么原因心里明明非常清楚，还非要说出来。

信用卡多了还真是不方便，六张信用卡加上支付宝花呗，我算是有七个消费渠道了，还款时也挺折磨人的，真想注销掉几张信用卡。

一直想找个费率低又好用的证券应用，但是一直遇不到满意的，海通证券费率是超低，App 也还行，但是有两点让我非常不爽：一是每个周末都要维护，无法使用，二是每次手机杀后台之后都需要重新输入密码登录。

更新：终于找到了一个费率和海通证券一样，甚至某些特殊品种比海通证券费率还低的银河证券。银河证券有个条件单是很不错，可惜依然没有定投条件单，App 功能不如海通证券，但是相对流畅一些。就当备用了，国庆回来注销掉长江证券，因为它没有吸引我的地方了。

最终我还是选择了更换手机号，这两天更换了五十多个产品绑定的手机号，之后还会有更多，但是我需要慢慢更换了。

一直都很期待联通的异地跨域融合，彻底打破地域限制，可是结果却让人很不满意，跨域融合需要那种全国性的套餐，明显比本地套餐更贵，这样一来跨域融合就没啥意义了。

前几天还想携号转网到中国移动，但是由于移动无法异地办理携号转网，我也就没在坚持。虽然联通还是有各种小毛病，但是整体服务还算可以，很多事情都是走在前列。

昨天才知道工银亚洲已经免管理费一年了，看来这个银行卡真香，可惜需要找时间更新一下港澳通行证，依然得去香港一趟。

海通证券的 IT 基础设施是真垃圾，用的越久越会发现问题。需要一个强有力的技术领导者好好把海通的 IT 基础设施改造一下。

很多证券简称都改名很久了，海通里还是旧简称。我一直以为是海通数据没同步，多刷了几次后我发现有时名字是新的有时是旧的，这说明他们的后端服务没完全同步啊，一些是新数据一些是就数据。

如果有其他费率和海通一样低的app，我真要弃海通而去了。

真想不到有一天我居然不在余额宝里放钱了，现在钱都是放在平安银行T+0理财里面，可以转账和支付使用，方便的很。

申请了个中国银行的 Visa 信用卡，来替代招商银行的 Visa 信用卡用作外币支付。

自此，广发、中信、招商的信用卡彻底被我冷冻了，以后主刷工行、中行和建行的信用卡。

开了银河证券账户还学到了新知识：基金套利、基金跨系统托管和基金系统内托管。

前天晚上看到平安期货永久免佣金的活动，加上我现在挺喜欢平安的，于是就开了个账户，今天把佣金调整好了。这三天一直在学习期货、期权知识，在适当的时候期货、期权确实能放大收益。国庆回来等我腾出资金了，就先把期货账户的期权交易给开开玩玩，看起来不错的样子。

之前有段时间研究了一下美股免佣，感觉微牛证券挺不错的。最近发现微牛证券支持港股交易了，果断开户一下。

『工欲善其事，必先利其器』，在证券和期货市场也是如此，经过这段时间的折腾，我感觉我在证券和期货市场的『器』已经准备好了，接下来的关注点就应该放在『事』上面了。

A 股：海通证券、银河证券，佣金都非常非常低，主用海通证券（科创板、港股通、两融、期权都是在海通证券）。
港美股：微牛证券、富途证券，富途证券非常好，但是微牛证券费率更优，我慢慢的会把富途里面的钱都转到微牛，最终就只用微牛证券了。
期货交易：平安期货免佣挺有吸引力的，App 也还可以，所以就用它了。 我现在用创元期货，并且暂时没有换期货商的打算。

第一次做期货交易，这个投机太严重，一不小心就资金归零了，得稳住。

期货太刺激了，心里真的是止不住的激动，心态还是差一些。这些东西还是得亲身经历了才能慢慢的适应并看淡。

基金系统内转托管成功了，以后慢慢的把基金合并到一个平台上，便于管理。

所谓场外基金的系统内转托管就是把基金从一个销售平台转移到另一个销售平台。比如把蛋卷基金转移到支付宝。

我现在每天在期货、A股、港股、美股等七八个应用里切来切去，实在太折腾了。需要保持专注，期货这东西得看确切的机会，刷单这事我做不来。
